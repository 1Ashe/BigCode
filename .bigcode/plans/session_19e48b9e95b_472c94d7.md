# 计划：创建、执行并删除 Python 文件

## 步骤

1. **创建 Python 文件** `/home/qt/tmp/hello.py`，内容为：
   ```python
   print("hello world")
   ```

2. **执行该文件**：
   ```bash
   python3 /home/qt/tmp/hello.py
   ```

3. **删除该文件**：
   ```bash
   rm /home/qt/tmp/hello.py
   ```

## 说明
- 这是一个简单的三步操作：创建 → 运行 → 清理。
- 需要 `/home/qt/tmp/` 目录存在，如不存在需先创建。
